# 牙牙消息
一个口袋48电脑端工具

## 主要功能

- 口袋房间：查看关注成员的口袋房间消息。
- 抓取消息：抓取成员房间消息并一键导入至软件内。
- 消息检索：根据关键词查找成员房间中的历史消息，可根据日期及消息类型进行筛选。
- 数据统计：统计成员房间每日消息条数、互动次数、翻牌相关数据。
- 公演记录：查询成员公演记录，观看历史公演。
- 成员档案：查询成员详细资料和公式照。
- 房间相册：查看成员房间图片、视频，支持批量下载。
- 数据库：48成员相关数据。
- 直播：观看成员直播，支持送出直播间礼物。
- 回放：观看、下载成员直播回放。点击弹幕/字幕时间轴可跳转对应时间点。
- 切片：直播实时切片、录播回放切片、公演切片。
- 翻牌：向成员进行翻牌提问，查看账号历史翻牌。支持撤回提问、删除翻牌。
- 电台：收听、录制成员房间上麦。
- 资源：官方音乐、电台、视频资源。

## 说点别的

为什么要叫牙牙消息呢？因为本来只是太无聊了想看看牙以前发了什么消息，奈何直接下载出来的html文件看的实在是不太方便，于是想着写一个能检索消息的工具，方便我查找牙在什么时候发了什么消息。之后突发奇想，为了方便在电脑上看直播，于是加入了直播和回放，纯属意外。

anyway，感谢使用。
