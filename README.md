# 🦷 牙牙消息 (yaya msg)

> 一个用于查看口袋48直播/回放，以及检索口袋房间消息的桌面端工具。

## 📖 简介

本工具主要功能包括：
- **消息检索**：方便快捷地根据关键词查找成员的历史房间消息。可根据日期及消息类型进行筛选。
- **数据统计**：统计成员每日消息条数及与聚聚互动次数。
- **直播/回放**：支持直接调用播放器观看口袋48的直播与录播回放。

## 🛠️ 常见问题：无法调用 PotPlayer 播放

如果您点击头像跳转后无法直接播放，且显示以下提示：
> “如果 PotPlayer 没有弹出，说明您的 URI 协议未注册或被拦截。请手动复制链接后在 PotPlayer 中打开”

请按照以下步骤手动注册 URI 协议：

### 1. 下载 URLProtocol 工具

| 平台 | 链接 |
| :--- | :--- |
| **GitHub** | [xifangczy/URLProtocol](https://github.com/xifangczy/URLProtocol/releases/tag/0.4) |
| **百度网盘** | [点击前往下载页](https://pan.baidu.com/s/18bPJH1f6BoSOSjaCijrNIw?pwd=0403) |

### 2. 配置步骤

1. 打开下载好的 `URLProtocol` 工具。
2. **协议名** 填写：`potplayer`
3. **目标程序** 选择：您的 PotPlayer 安装路径（即 `PotPlayerMini64.exe` 或 `PotPlayerMini.exe` 所在位置）。
4. 点击 **“添加/更新”** 按钮。

完成上述步骤后，即可正常跳转播放。

之后突发奇想，为了方便在电脑上看直播，于是加入了直播和回放，纯属意外。
anyway，感谢使用。
